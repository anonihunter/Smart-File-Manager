# Smart File Organizer

A Python project that automatically organizes files into categorized folders and detects duplicate files.

## Features

- Organizes files based on their extensions.
- Supports Documents, Images, Videos, Audio, and Archive files.
- Automatically creates destination folders if they don't exist.
- Detects duplicate files.
- Built using Python's `pathlib` and `shutil` modules.

## Project Structure

- Phase 1: File Organization ✅
- Phase 2: Duplicate File Detection ✅
- Phase 3: Folder Monitoring 🚧 (In Progress)

## Technologies Used

- Python
- pathlib
- shutil
- hashlib

## How to Run

1. Clone the repository.
2. Update the target directory path in the script.
3. Run the Python file.
